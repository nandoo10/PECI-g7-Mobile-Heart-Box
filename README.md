# Projeto-Final-de-Licenciatura

O projeto Mobile HeartBox propõe um sistema de telemetria desportiva e monitorização de saúde baseado numa arquitetura IoT. Ao contrário de ecossistemas comerciais fechados, esta solução permite a recolha, transmissão e armazenamento de dados biométricos (BPM, temperatura corporal) e dados de contexto (GPS, deteção de quedas, proximidade de obstáculos), garantindo ao utilizador o controlo total e acesso bruto ao seu histórico. O sistema combina hardware, com uma infraestrutura Cloud escalável para entregar monitorização em tempo real e análise histórica.
